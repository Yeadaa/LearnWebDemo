### 一、git的常用命令

##### 1. 将远程仓库拉取到本地使用 `git clone [仓库地址]` 命令

例如： `git clone git@github.com:Yeadaa/LearnWebDemo.git`



##### 2. 添加文件使用 **vscode** 的 git 工具。

![image-20190716230757300](http://ww4.sinaimg.cn/large/006tNc79gy1g522yuuqprj30ka0esgnc.jpg)

例如： ` git push`
